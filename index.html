О проекте

Анонимность нового поколения

Tanegi — это децентрализованная P2P-сеть с луковой маршрутизацией, созданная полностью на Rust. В отличие от Tor, Tanegi использует пост-квантовый KEM (Kyber-1024) и собственный математический алгоритм, основанный на задаче числа 10958.

Протокол Double Ratchet (как в Signal) обеспечивает совершенную прямую секретность в чате. Kademlia DHT организует распределённую хранилищную систему с расширенным кодированием Reed-Solomon.

⚠
Beta-версия. Проект находится в стадии разработки. Не использовать для защиты критически важных данных в production.
tanegi-core / network / handshake
let keypair = X25519::generate(&mut OsRng);
let kyber = Kyber1024::keygen();
let shared = HybridKEM::encapsulate(
  &peer_pub, OrderMode::Random
); // ChaCha-first or Kyber-first
 
let ratchet = DoubleRatchet::init(shared);
// Forward secrecy from message 1
 
let circuit = OnionBuilder::new()
  .hops(3..7)
  .subnet_diversity("/16")
  .build()?; // anti-Sybil
 
✓ Circuit established — 5 hops

Приватность

Нулевая утечка

Многоуровневая защита от деанонимизации на каждом уровне стека.

🔥
Kill Switch
iptables автоматически блокирует весь трафик при разрыве VPN-туннеля. Требует root. Защищает от IP-утечек.
🎭
Cover Traffic
Адаптивная генерация фиктивного трафика (10% пропускной способности). Защита от traffic analysis.
🚫
Блокировка трекеров
DNS-блокировка google-analytics, doubleclick, facebook, telemetry Microsoft/Mozilla и 20+ других доменов слежки.
📡
DoH (DNS over HTTPS)
RFC 8484. Все DNS-запросы зашифрованы. Fallback — DNS через Tanegi-цепочку. Нет DNS-утечек.
📹
WebRTC Blocking
iptables правила блокируют WebRTC STUN/TURN запросы, которые могут раскрыть реальный IP-адрес.

Запуск за 3 шага

1
Скачайте и распакуйте
Скачайте tanegi.zip, распакуйте в удобное место.
unzip tanegi.zip && cd tanegi/
2
Сделайте исполняемым
Бинарный файл требует права на исполнение и запуск от root для работы с iptables.
chmod +x tanegi && sudo ./tanegi
3
Настройте SOCKS5 прокси
Укажите приложениям (браузер, curl) использовать SOCKS5 прокси 127.0.0.1:9050.
curl --socks5-hostname 127.0.0.1:9050 https://check.torproject.org
+
Дополнительные флаги
Включите усиленную защиту приватности.
sudo ./tanegi --tui --block-trackers --cover-traffic
⚠
Для работы kill switch и iptables-правил требуются права root (Linux). На macOS/Windows некоторые функции недоступны.
